# Maps (`05_maps`)

**Purpose:** Cross-chapter **braiding** and dependency views. These are **working** maps (PROPOSED allowed); authoritative dates and facts still promote through `writing/01_canon/`.

## How to use

- One row per **thread**, **reveal**, **symbol**, **location braid**, or **dependency edge** — not prose.
- Link each row to chapter folders under `writing/03_epics/…/chapters/ch_###_<slug>/` and to `source_matches/` where relevant.
- When a map row becomes canon-stable, mirror the fact in `01_canon/master_*.md` and mark the map row `CONFIRMED` with a pointer.

## Files

| File | Tracks |
|------|--------|
| `master_timeline_map.md` | In-world ordering vs chapter order |
| `character_arc_map.md` | Character pressure across chapters |
| `emotional_arc_map.md` | Reader emotional beats across chapters |
| `symbol_braid_map.md` | Motifs / objects — introduce, vary, pay off |
| `setting_braid_map.md` | Places and environmental through-lines |
| `history_braid_map.md` | Lore / public vs secret history threads |
| `secret_reveal_map.md` | Who knows what, when revealed |
| `family_lineage_map.md` | Lineage / household edges affecting plot |
| `narrator_knowledge_map.md` | Narrator vs reader vs character information |
| `chapter_dependency_map.md` | Hard ordering / causal prerequisites between chapters |
| `theme_braid_map.md` | Thematic pressure and moral questions across chapters |
