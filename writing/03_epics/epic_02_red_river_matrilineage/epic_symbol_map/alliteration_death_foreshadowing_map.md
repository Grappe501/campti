# Alliteration / Death Foreshadowing Map

Canon status: PROPOSED

## Human direction

Alliteration should play a hidden role throughout the novels and the entire epic.

Before any sign of death, before the first direct foreshadow, thought, image, or mention that leads toward death, back up approximately ten paragraphs and place a whole paragraph containing noticeable but seemingly natural alliteration.

The alliterative paragraph may appear in:

- conversation
- song
- nature description
- work description
- prayer-like rhythm, if culturally safe
- a child’s chant
- a joke
- a repeated work phrase
- a description of weather, insects, leaves, water, tools, animals, or labor

The reader should not consciously know it is a death signal.

## Purpose

Alliteration becomes a hidden death-symbol.

The reader may later feel something approaching before understanding why.

## Rule

Alliteration must be:

- natural
- beautiful
- placed with purpose
- not cartoonish
- not tongue-twister obvious
- not always the same sound
- not repeated so often it becomes mechanical

## Sound families

| Sound type | Possible mood | Use carefully |
|---|---|---|
| s / sh | softness, smoke, secrecy, slipping, silence | can become too obvious |
| b / p | blunt force, bodies, breath, breakage | use for impact |
| r | river, rush, unrest, rolling | good for Red River movement |
| f | flame, fear, feather, flesh, forgetting | strong death-adjacent potential |
| m | memory, mother, mud, mourning | grief / matriarchal use |
| hard c / k | cut, crack, cold, command | violence / rupture risk |
