# Metaphor To Plot Map

Canon status: PROPOSED

## Purpose

Link **earned metaphors** to **plot engines** so imagery does not float beside story.

| Metaphor (earned) | Plot engine it feeds | Example pressure | Book band | Status |
|---|---|---|---|---|
| Binding cord | marriage / obligation / teaching | false security before raid | 01 | PROPOSED |
| River current | trade route / escape / law | who may cross; who profits | 01–05 | PROPOSED |
| Gumbo pot (late) | blended inheritance / civic persona | public vs private kitchen | 08–09 | PROPOSED |
| Injury as public body | marshal myth / private cost | temper + partnership | 08 | OPEN |
| Silence as memory | withheld testimony | women carry what records omit | 01–09 | PROPOSED |
| Color return (shifted) | moral reversal / time jump | same hue, new cost | TBD | OPEN |

## Rule

If a metaphor does not **force a choice**, **expose a cost**, or **transfer power**, it is not ready for recurrence.
