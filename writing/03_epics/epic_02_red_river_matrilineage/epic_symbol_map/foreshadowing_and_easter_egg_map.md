# Foreshadowing And Easter Egg Map

Canon status: PROPOSED

## Purpose

Plant **fair** echoes: details that matter later must be **true in the moment**, not tricks.

## Rules

- **Foreshadowing:** information appears because the scene needs it now; later it **re-prices** meaning.
- **Easter eggs:** small, optional rewards for attentive readers; must not gate comprehension.
- **Forbidden:** “little did she know” machinery; repeated wink phrases.
