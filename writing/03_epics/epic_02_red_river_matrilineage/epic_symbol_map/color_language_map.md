# Color Language Map

Canon status: PROPOSED

## Purpose

Track **color as intentional language** — mood, politics, material truth — without turning prose into a paint-by-numbers thesis. The **full season + palette system** lives in `color_season_symbol_system.md`; this file stays the **roll-up tracker** for anchored color beats.

## Rules

- **First use:** tie color to **material** (mud, fish scale, cloth dye, weather) not abstraction.
- **Return:** only when **meaning shifts** (see `../language_doctrine/information_once_rule.md`).
- **Avoid:** billboard symbolism (“red for anger”) unless the scene earns it through action.
