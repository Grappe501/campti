# Silence And Unsaid Map

Canon status: PROPOSED

## Purpose

Track **what is not said**, **who cannot speak**, **who refuses**, and **what silence protects** — across books.

## Types of silence

| Type | Story function | Risk |
|---|---|---|
| Protective silence | shields child / community | reader confusion if unmotivated |
| Shame silence | hides violation / fear | melodrama if over-signaled |
| Power silence | command without speech | tyranny misread as mysticism |
| Grief silence | unspeakable loss | bathos |
| Strategic silence | legal / survival | must pair with consequence |
