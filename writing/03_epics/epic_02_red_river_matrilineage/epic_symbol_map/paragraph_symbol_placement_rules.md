# Paragraph symbol placement rules

Canon status: PROPOSED

## Purpose

Track **where** symbolic density is allowed to cluster so prose stays intentional — without “symbol every sentence.”

## Rules

1. **First paragraph:** see `../language_doctrine/chapter_opening_paragraph_doctrine.md` — highest craft load; contains tone, breath, seeds, secret argument.
2. **Turn paragraphs:** allow symbol **re-pricing** (meaning shifts) at scene pivots — not at every pivot.
3. **Death-adjacent passages:** optional alliteration “pressure valve” paragraph ~10 paragraphs before death-toward material — see `alliteration_death_foreshadowing_map.md`.
4. **Closing beat:** may echo opening **only** if echo **changes** meaning (reader returns wiser).

## Anti-rules

- No symbol stacking in exposition disguised as description.
- No “meaning summary” final paragraph unless the character would honestly think it (rare).
