# Color and season symbol system

Canon status: PROPOSED

## Purpose

Colors and seasons should carry mood, era, power, and hidden structure across the epic.

## Rule

Do not make color symbolism obvious.

Color should work as atmosphere, object, clothing, weather, food, light, soil, blood, paper, metal, ash, river, or political world.

## Color map

| Color | Possible meaning | Era / book use | Character link | Risk |
|---|---|---|---|---|
| red | river, blood, clay, power, warning, inheritance | 01–09 | Red River / Grappe line | too obvious if always danger |
| black | soil, ash, roux, night, grief, law, hidden memory | 01–09 | gumbo / mourning / record gaps | over-darkening |
| white | bone, paper, French records, salt, cloth, public respectability | 02–09 | colonial/legal world | simplistic purity symbolism |
| green | sassafras, growth, healing, seasons, children | 01–09 | women / food memory | too soft |
| blue | distance, water, uniforms, politics, sky | 06–09 | war / public life | avoid generic sadness |
| gold / yellow | corn, firelight, age, heat, ambition | 01–09 | household / harvest | too decorative |
| gray | smoke, age, fog, compromise, Reconstruction ambiguity | 01–09 | transitions | overused gloom |
