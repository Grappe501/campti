# Silence As Language

Canon status: PROPOSED

## Purpose

Silence is not empty space; it is **information under pressure**.

## Types (operator)

- **Refusal to answer** (power)
- **Cannot find words** (fear, grief)
- **Strategic pause** (negotiation)
- **Shared silence** (trust or dread)

## Rules

- Pair silence with **physical continuation** (hands still working) so scenes do not stall.
- Track long threads in `../epic_symbol_map/silence_and_unsaid_map.md`.
