# No Overexplaining Rule

Canon status: PROPOSED

## Rule

If a line exists mainly to **translate** the symbol for the reader, delete it.

## Signals of overexplaining

- “This meant…” / “It was as if…” / “She knew this represented…”
- Double sentences saying the same emotional fact.
- Moral summaries at the end of otherwise physical scenes.
