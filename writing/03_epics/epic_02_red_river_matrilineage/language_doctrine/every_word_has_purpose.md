# Every Word Has Purpose

Canon status: PROPOSED

## Governing rule

Everything should carry meaning:

- color
- silence
- nature
- pacing
- sentence breath
- emotional mood
- object choice
- repeated image
- absence
- injury
- food
- weather
- humor
- body language

No filler.

## Test

For any paragraph, ask:

1. Does it build world?
2. Does it reveal character?
3. Does it move power?
4. Does it deepen symbol?
5. Does it plant or echo an arc?
6. Does it control breath or mood?
7. Does it earn its place?

If not, cut or revise.

## Opening paragraphs

For the **first paragraph of every chapter**, apply the stricter craft bar in `chapter_opening_paragraph_doctrine.md` (tone, breath, hidden seeds, secret argument) **in addition** to this test.
