# Chapter Pace Design

Canon status: PROPOSED

## Purpose

Track the emotional rhythm, breath, density, and movement of chapters.

## Pace types

| Pace type | Function | Use |
|---|---|---|
| River pace | slow, immersive, flowing | world-entry, memory, landscape |
| Work pace | rhythmic, tactile, embodied | women’s labor, household power |
| Trade pace | alert, observant, social | visitors, goods, pressure |
| Binding pace | anticipatory, gathering | wedding/binding movement |
| Conflict pace | sharp, compressed | marital fights, leadership tests |
| Raid pace | fragmented, sensory, child-terror | final collapse only |
| Aftermath pace | withheld / silent / controlled | later books |
