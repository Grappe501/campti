# Sassafras / Filé / Gumbo Across Centuries

Canon status: PROPOSED pending research

## Governing symbol

Sassafras begins as **Native plant knowledge** and household **food/medicine** knowledge.

Across the epic, it becomes **filé**, **gumbo**, **memory**, **family identity**, and living proof that old knowledge still feeds the present.

## Human family memory

The tree where the human’s grandfather and great-grandfather picked the best sassafras leaves is still being picked today to make filé for gumbo.

This living practice should anchor the symbol.
