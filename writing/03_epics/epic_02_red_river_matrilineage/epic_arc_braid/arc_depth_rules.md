# Arc Depth Rules

Canon status: PROPOSED

## Rules

1. **Earn collisions** — especially late-series *Crash*-style convergence; no single chapter “explains the epic.”
2. **Each book pays rent** — a complete human story + an inheritance debt to the next book.
3. **Subplots sleep** — absence is allowed; return must increase cost.
4. **No power without cost** — public hero arcs extract private payment (see injury + marriage braids).
5. **Research + ethics gates** for historical violence, Native specificity, and modern public figures.

## Pairing

- Arc beats should **mate** with symbols through `../epic_symbol_map/metaphor_to_plot_map.md`.
