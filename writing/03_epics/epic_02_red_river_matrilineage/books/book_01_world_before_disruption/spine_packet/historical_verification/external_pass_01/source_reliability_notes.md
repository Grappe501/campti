# Source reliability notes

Canon status: PROPOSED

## Source types (EXT-HIST-001 — 004)

| Type | Strength | Weakness |
|------|----------|----------|
| Tribal/community history page | Community authority, narrative coherence | May synthesize oral + compiled history; needs dates corroborated |
| Federal acknowledgment petition | Extensive compilation; useful bibliography leads | Advocacy frame; fact-check each claim |
| Secondary overview (e.g. 64 Parishes) | Good for **separating** commonly confused events | Not primary; can lag specialist corrections |
| Methodology (Evidence Explained style) | Prevents false precision | Does not supply dates |

## Discipline (from EXT-HIST-004)

- **PROPOSED:** Prefer **church / civil / military** record trails over merged genealogy trees for dates and identities.
- **OPEN:** Build a **minimum citation set** per row in `candidate_event_matrix.md` before any canon promotion.
