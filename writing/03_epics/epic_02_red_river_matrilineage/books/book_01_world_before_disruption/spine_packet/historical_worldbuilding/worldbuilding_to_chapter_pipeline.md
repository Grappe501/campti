# Worldbuilding to Chapter Pipeline

Canon status: PROPOSED

## Rule

No chapter should be outlined until it satisfies **this** checklist **and** the **core storyline** requirements in `../core_storyline/` (including `../core_storyline/storyline_to_chapter_manifest_gate.md` before a real chapter manifest).

## For Each Future Chapter, Identify

- village life function
- family life function
- child-world function
- trade route function
- historical pressure function
- relationship function
- sensory world
- cultural verification needs
- what the reader comes to love
- what future the chapter makes feel possible
- **core storyline** function (visible binding arc, hidden Red River pressure, **HI-004** matriarch / mother-gen / **granddaughter** beat — see `../core_storyline/`)
