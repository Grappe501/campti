# Chapter Folder Creation Plan

Canon status: PROPOSED

## Rule

This file prepares the future creation of chapter folders. It does not authorize creation.

Do **not** promote draft prose into `writing/01_canon/` from this step alone.

---

## If Approved

Create **one folder per approved unit** under the Book 01 working tree (exact parent folder name follows manifest lock — e.g. a `chapters/` directory beside `draft_lab/`, **not** final canon).

Example shape (15 folders):

- `ch_001_<slug>/`
- `ch_002_<slug>/`
- … through `ch_015_<slug>/`

See **`recommended_chapter_folder_names.md`** for proposed slugs. Numbering assumes sequential `ch_001` … `ch_015` maps to UNIT-001 … UNIT-015 after human approval (renumber if merge/split later).

Each folder should be **cloned** from:

`writing/00_system/folder_blueprints/chapter_blueprint/`

At clone time, **align** required filenames with whatever the blueprint currently ships (e.g. `chapter_brief.md`, `outline.md`, `scenario_board.md` exist today; add `chapter_contract.md` / `chapter_exit_condition.md` if you adopt those names in the blueprint).
