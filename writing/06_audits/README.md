# Audits (`06_audits`)

**Purpose:** Repeatable **checklists** for correctness, traceability, and braiding. Record pass/fail + notes; link to chapter `continuity/`, `05_maps/`, and `01_canon/`.

## How to run

- Run the relevant audit when a chapter hits `Continuity pass` or `Production pass` on the dashboard (`writing/04_production_dashboard/chapter_status.md`).
- Log outcomes in the audit file (append a dated section) or in the chapter’s `production/quality_pass.md`.

## Files

| File | Focus |
|------|--------|
| `continuity_audit.md` | Facts, dependencies, thread carry |
| `timeline_audit.md` | Dates, ordering, compression |
| `character_voice_audit.md` | Diction, rhythm, drift |
| `pov_audit.md` | POV rules, leaks, camera |
| `canon_label_audit.md` | CONFIRMED vs PROPOSED discipline |
| `unresolved_questions_audit.md` | OPEN items surfaced vs hidden |
| `symbol_repetition_audit.md` | Echo vs clutter |
| `chapter_purpose_audit.md` | Chapter contract vs outcome |
| `pacing_audit.md` | Beats, breath, density |
| `historical_integrity_audit.md` | Claims vs sources |
| `emotional_logic_audit.md` | Motivation / feeling coherence |
