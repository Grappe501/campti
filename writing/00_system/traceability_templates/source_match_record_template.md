# Source match record (template)

**One record = one chunk ↔ one chapter (or one scene).** Copy a block per match.

| Field | Value |
|-------|--------|
| Record ID | SM-### |
| Chapter | `ch_###_<slug>` |
| Scene (optional) | `scene_###` |
| Source path | `writing/02_source_chunks/...` |
| Chunk index row | (line or slug in `chunk_index.md`) |

## Match meta

| Field | Value |
|-------|--------|
| Use case | exposition / beat / voice / lore / sensory / other |
| Fit | high / med / low |
| Canon status of source | unknown / draft / confirmed |
| Label for use in prose | CONFIRMED / PROPOSED |

## Excerpt pointer

- Start anchor: (phrase / paragraph id)
- End anchor:
- Why this excerpt: (one line)
