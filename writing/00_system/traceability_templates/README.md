# Traceability templates

Copy rows or whole files into chapter folders (e.g. `chapter_contract.md`, `chapter_exit_condition.md`) or append dated blocks to audit / map files in `writing/05_maps/` and `writing/06_audits/`.

| Template | Use |
|----------|-----|
| `chapter_contract_template.md` | Lock what the chapter must accomplish (traceability). |
| `chapter_exit_condition_template.md` | Define acceptable end state + carry + review flags. |
| `scene_card_template.md` | One scene’s structural card. |
| `source_match_record_template.md` | One chunk ↔ chapter linkage row. |
| `braid_map_table_template.md` | Generic braid row shape for `05_maps/*.md`. |
| `character_knowledge_record_template.md` | Who knows what, chapter boundary. |
| `canon_decision_record_template.md` | Promotion / demotion log. |
| `contradiction_record_template.md` | Structured contradiction ticket. |

Canon labels: `writing/00_system/canon_status_labels.md`.
