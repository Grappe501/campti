# Canon status labels

Use these labels consistently across `writing/` (and anywhere else you track story truth).

| Label | Meaning |
|-------|---------|
| **CONFIRMED** | Matches `01_canon/` or an explicit team decision recorded there. |
| **PROPOSED** | Speculative, inferred, or draft-only — **not** authoritative until promoted. |
| **OPEN** | Unresolved question or fork; see `01_canon/unresolved_questions.md` or epic/book question files. |
| **DEPRECATED** | Formerly used; kept for audit trail — do not build new work on it without review. |

## Rules

- Nothing becomes canon **silently**. Promotion = edit `01_canon/` + remove or relabel downstream `PROPOSED`.
- If material is **inferred** from sources, mark **PROPOSED** until verified against research or canon locks.
- When pasting AI output, preserve or add these prefixes; do not strip uncertainty to make copy read smoother.

See also: `naming_conventions.md`, `04_production_dashboard/open_questions_status.md`.
