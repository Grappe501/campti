# Production targets

Roll-up status for the manuscript lives in **`writing/04_production_dashboard/`** (see `chapter_status.md`).

## What “~75% draft” means

A chapter at **~75% draft** is not polished prose. It is **structurally sound** and **narratively legible**:

- **Brief and outline** agree with the scene list; no orphan beats.
- **Scenario board** has been culled into a deliberate scene sequence.
- **Source matching** has been attempted; gaps are listed, not hidden.
- **Draft** carries POV, intent, and turning points clearly enough that a line editor can work without guessing plot function.
- **Continuity** documents new facts, dependencies, and any tension with existing canon (with PROPOSED vs CONFIRMED).
- **Production pass** lists only actionable remaining work (research, canon decision, new scene, etc.).

## Stage gates (suggested)

| Stage | Brief | Outline | Scenario | Sources | Scene list | Draft | Continuity | Production |
|-------|:-----:|:-------:|:--------:|:-------:|:----------:|:-----:|:----------:|:----------:|
| Skeleton | ✓ | ✓ | partial | partial | outline only | empty / stubs | threads only | missing_assets full |
| Scaffold | ✓ | ✓ | ✓ | ✓ started | ✓ | partial | open_threads | quality_pass draft |
| ~75% draft | ✓ | ✓ | ✓ | ✓ | ✓ | substantive | ✓ updated | quality_pass tightening |
| Polish-ready | locked* | locked* | archived* | ✓ | ✓ | complete pass | resolved or escalated | minimal gaps |

\*“Locked” means changes require explicit revision notes, not that files are immutable.

## Book and epic alignment

- **Book**: `book_chapter_manifest.md` reflects each chapter’s stage.
- **Epic**: `epic_questions.md` tracks cross-book unknowns; promote closed items to canon when appropriate.

## Anti-goals at this milestone

- Final line-level voice polish across the whole manuscript.
- Premature merging of all subplots into a single simplified spine.
- Silent canon invention without `PROPOSED:` or canon promotion.
