# Scoring Rubric

Canon status: PROPOSED

## Progress Score Meaning

| Score | Meaning |
|---:|---|
| 0 | Not started |
| 10 | Folder / placeholder exists |
| 25 | Scaffold exists |
| 40 | Evidence/source gathering started |
| 55 | First fill pass complete |
| 70 | Reviewed/revised once |
| 85 | Human-selected or accepted as provisional model |
| 95 | Ready for lock / chapter / canon / drafting decision |
| 100 | Locked for current phase |

## Status Labels

Use these status labels:

- NOT STARTED
- SCAFFOLDED
- IN PROGRESS
- FILLED V0
- REVIEWED
- REVISED
- HUMAN SELECTED
- PROVISIONAL MODEL
- BLOCKED
- READY FOR REVIEW
- READY FOR NEXT PHASE
- LOCKED FOR CURRENT PHASE

## Canon Warning

Progress does not equal canon certainty.

A section can be 85% structurally developed and still remain `PROPOSED` or `OPEN` historically.

Always track:

- progress score
- canon status
- blocker status
- next action
