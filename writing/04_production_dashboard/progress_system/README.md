# Progress System — KPI / bar charts

Canon status: PROPOSED

## Purpose

Human-readable **progress bars and KPI tables** from **epic → book → arc/character/symbol/research → prose → movement/chapter readiness**, without implying canon lock.

**Not included:** automatic graph rendering (Markdown-only). Optional: import `data/*.csv` into a spreadsheet for real charts.

## Rules (short)

- **20-block bar**; score meanings in `scoring_rubric.md`.
- **100%** = complete for the **tracked phase**, not publication-final.
- **Progress ≠ canon truth** — keep `PROPOSED` / `OPEN` visible in narrative docs.
- Do **not** raise scores only because a file exists (`update_rules.md`).
