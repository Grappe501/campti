# KPI Definitions

Canon status: PROPOSED

## Epic-Level KPIs

| KPI | Description |
|---|---|
| Series spine | Books 01–09 architecture, role locksheet, transfer map |
| Historical anchor | documented raid/person/lineage anchors |
| Women-centered arc | matriarchal continuity across books |
| Book map | each book’s era, purpose, woman center, male interface |
| Research foundation | source hierarchy, blockers, verification paths |
| Prose doctrine | style, scale, pacing, scene texture |
| Canon discipline | labels, decisions, contradiction tracking |

## Book-Level KPIs

| KPI | Description |
|---|---|
| Book spine | governing structure and ending condition |
| Core storyline | visible plot, hidden plot, emotional plot |
| Historical worldbuilding | trade, village life, kinship, foodways, seasonal systems |
| Character architecture | primary figures, family relationships, reader attachment |
| Arc architecture | matriarchal, binding, Red River pressure, survival |
| Symbol/theme architecture | recurring objects, motifs, inheritance symbols |
| Manifest | thirds, movements, future chapters |
| Movement packets | prose-ready story packets |
| Prose lab | voice tests, samples, revisions |
| Research blockers | open factual or cultural questions |

## Movement / Chapter-Level KPIs

| KPI | Description |
|---|---|
| Purpose | movement/chapter job defined |
| Primary woman | matriarchal center identified |
| Power transfer | what power is built, gained, lost, transferred |
| Visible plot | what the reader thinks is happening |
| Hidden pressure | what historical pressure exists under the surface |
| Character attachment | who the reader loves more by the end |
| Setting texture | place, nature, material culture, labor |
| Research flags | uncertainty marked |
| Exit condition | what has changed by the end |
| Prose readiness | ready to draft or not |

## Character KPIs

| KPI | Description |
|---|---|
| Public role | what the world sees |
| Private desire | what the character wants |
| Wound/fear | what pressure shapes them |
| Power function | what power they carry |
| Relationship web | family, rivals, allies, obligations |
| Historical source status | documented / proposed / fictionalized |
| Reader attachment | why the reader cares |
| Arc status | how they change |

## Symbolism KPIs

| KPI | Description |
|---|---|
| Symbol identified | object/motif exists |
| Meaning defined | what it carries emotionally/thematically |
| Character connection | who it belongs to |
| Recurrence plan | where it returns |
| Transformation | how meaning changes |
| Ending echo | whether it matters later |
