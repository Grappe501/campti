# Progress Update Rules

Canon status: PROPOSED

## When To Update Progress

Update charts after:

- new human input
- new synthesis version
- new manifest fill pass
- new prose sample
- review gate completion
- research verification
- canon decision
- movement packet fill
- chapter candidate creation
- chapter draft creation

## Rule

Never raise a score just because a file exists.

A file existing usually means 10–25%.

A file filled with evidence and reviewed may reach 55–70%.

Human-selected direction can reach 85%, but only for story direction, not historical certainty.

## Required Fields

Every chart row should include:

- score
- bar
- status
- blocker
- next action

## Snapshot Log

Every major update should append a dated entry to `progress_snapshot_log.md`.
