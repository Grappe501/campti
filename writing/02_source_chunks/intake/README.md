# Source intake (`02_source_chunks/intake/`)

**Purpose:** Inbox for **existing** story / world / research material **before** it is sorted into typed folders under `writing/02_source_chunks/`.

## Flow

1. Drop notes, pasted excerpts, or links into `new_unsorted_material.md` **or** add new files and log them in `intake_log.md`.
2. Classify using `sorting_rules.md`.
3. Move content into `story_fragments/`, `worldbuilding/`, `characters/`, `settings/`, `history/`, `symbols/`, `data/`, or `unused_or_unsorted/` as appropriate.
4. Register in `../chunk_index.md`.
5. Update `writing/04_production_dashboard/source_sorting_status.md`.

## Relationship to `unused_or_unsorted/`

- **`intake/`** — active triage queue with a paper trail (`intake_log.md`).
- **`unused_or_unsorted/`** — longer-term holding when you are not ready to classify.

You can use either; prefer **`intake/`** when you want explicit logging for production.
