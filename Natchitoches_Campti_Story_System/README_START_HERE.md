# Natchitoches -> Campti Story System (Production Package)

This is a full research + writing operating system for Book 1, centered on Natchitoches continuity across Red River pressure and eventual Campti reformation.

## Core Narrative Law

The women of this line are carrying memory, obligation, and belonging across instability.  
Characters think in native relational cognition; English is the translation layer.

## How To Use In Cursor

1. Load the entire folder as project context.
2. Read in this order:
   - `01_WORLD_STATE_1650.md`
   - `02_ENVIRONMENT_SYSTEM.md`
   - `03_SOCIAL_STRUCTURE.md`
   - `04_MATRIARCH_DOSSIERS.md`
   - `05_LANGUAGE_COGNITION_SYSTEM.md`
   - `07_SENSORY_LIBRARY.md`
   - `08_CAMPTI_TRANSITION_ARC.md`
   - `06_CHAPTER1_ENGINE.md`
3. Do not draft prose until you lock:
   - season window
   - household duty map
   - one environmental signal
   - one social signal
   - one hidden pressure
4. Build scenes using `06_CHAPTER1_ENGINE.md` beat rules.

## Scene Build Protocol (Repeat Per Scene)

1. Select point-of-view carrier in the matrilineal line.
2. Define continuity task at risk.
3. Add one subtle wrongness (not catastrophe).
4. Show social read through behavior, not explanation.
5. End with unresolved pressure carry.
