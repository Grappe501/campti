# LANGUAGE + COGNITION SYSTEM

## Prime Rule

Characters think through relation, place, action, memory, and obligation.  
English is a translation surface, not the origin language of thought.

## Three-Layer Output Model

1. **Native cognition layer (internal)**
   - relation first
   - event and duty linked
   - place and memory intertwined
2. **Meaning mapping layer**
   - preserve logic order and consequence structure
3. **English render layer**
   - readable prose that keeps worldview orientation

## Default Thought Channels

- observation
- labor consequence
- kin obligation
- social reading
- bodily warning
- memory echo

## Translation Style Guidance

Avoid abstract self-analysis as default:

- "I am anxious."
- "She has unresolved trauma."
- "He is emotionally dysregulated."

Prefer embodied relational thought:

- "The basket is light too early."
- "He returns with no story where one should be."
- "The room changes when that name is spoken."

## Dialogue Guidance

- concise speech with high social load
- meaning often carried by timing, omission, and who answers whom
- avoid explanatory speeches about culture in active scenes

## Silence Semantics

Silence can mean:

- disagreement without rupture
- risk assessment in progress
- information protection
- status correction

Treat silence as data, not absence.

## Syntax and Rhythm Tips

- begin with perception or action before interpretation
- short clauses under pressure; longer clauses in reflection windows
- keep metaphor anchored to land/labor/kin rather than abstract philosophy

## Forbidden Drift

- modern clinical phrasing
- generic pan-Indigenous diction
- exposition-first cultural summarizing voice
- repetitive emotional labels detached from embodied context
