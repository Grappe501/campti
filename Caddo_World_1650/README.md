# Caddo World 1650 Research Package

This package contains structured research and narrative groundwork for Book 1, Chapter 1.

## Scope

This is a Natchitoches-centered Red River package focused on:

- continuity of people across displacement
- matrilineal and often matrilocal social order
- pre-spectacle chapter design ("order under pressure")
- native-language-first cognition design with English translation output
- long-arc movement toward Campti formation
